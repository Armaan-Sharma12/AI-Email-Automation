# AI-Email-Automation
Anyone can be overwhelmed by amount of email you get in a single day and it would be hectic to check every mail and mark those are important and delete those which are unnecessary; To make that easy I've made a Email Automation tool which checks your email every 15mintues and by the help of AI it auto reply those are important mail and if there's a deadline in the mail it automatically set it in your calendar.
For daily purpose, It would be life changing for every single person in this planet whose uses mail.
